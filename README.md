# Tierras Sagradas — Portal Web

Portal wiki mitológico construido con Next.js 14, Tailwind CSS, Framer Motion y Sanity CMS.

## Stack
- **Next.js 14** App Router
- **Tailwind CSS** con tema gold/obsidian
- **Framer Motion** para animaciones
- **GSAP** para animaciones avanzadas (próximamente)
- **Three.js** para efectos 3D (próximamente)
- **Sanity v3** CMS (projectId: ofky7gw5)

## Desarrollo local

```bash
npm install
npm run dev
```

Visita [http://localhost:3000](http://localhost:3000)

## Imágenes del Hero Slider

Las rutas de las imágenes del hero se configuran en:
`components/home/HeroSlider.tsx` — array `SLIDES` (líneas marcadas con `← CAMBIÁ ESTA RUTA`)

Las imágenes van en el directorio `public/images/hero/`.
